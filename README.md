## Description

A potentially hazardous object (PHO) is a near-Earth object ‚Äì either an asteroid or a comet ‚Äì with an orbit that can make close approaches to the Earth and is large enough to cause significant regional damage in the event of impact.
<br>
With asteroid simulator we can view latest asteroid approaches in real time.
